D.J. O'BRIEN AUTHOR PLATFORM V2

This package is a complete rebuild based on the organised CMS version.

Implemented:
- Separate Decap CMS sections for homepage, each trilogy book, characters, timeline, reviews, news, About, Contact and social links.
- Editable Page Scenery section with a separate epic-fantasy background for every public page.
- Local optimised WebP scenery, dark readability overlays and mobile handling.
- Custom domain references updated to https://djobrienauthor.com.
- SEO titles, descriptions, canonical URLs, social cards, sitemap, robots file and structured data.
- Netlify build script continues to combine separate CMS entries into content/site.json.

Deployment:
Upload the CONTENTS of this folder to the root of the GitHub repository and commit to main. Netlify will run npm run build automatically.
